# 변경 이력 (Changelog)

## v3.1.0 - 다국어 지원 및 창의적 디자인 차별화 (2025-01-29)

### 🌏 다국어(i18n) 시스템 구축

#### 새로운 기능
- **4개 언어 완전 지원**
  - 🇰🇷 한국어 (Korean) - 기본 언어
  - 🇺🇸 영어 (English)
  - 🇯🇵 일본어 (Japanese)
  - 🇨🇳 중국어 (Chinese)

#### 언어별 창의적 디자인 차별화
각 언어를 선택하면 국기 색상을 기반으로 테마가 자동 변경됩니다:

1. **한국어 테마**
   - 색상: 네이비 + 골드 (태극 색상)
   - Hero 배경: 빨강(#C60C30) + 파랑(#003478) 그라데이션
   - 느낌: 전통적이면서 프리미엄한 느낌

2. **영어 테마**
   - 색상: 블루 + 실버 (성조기 색상)
   - Hero 배경: 빨강(#B22234) + 파랑(#3C3B6E) 그라데이션
   - 느낌: 모던한 비즈니스 전문성

3. **일본어 테마**
   - 색상: 체리블라섬 핑크 + 화이트 (일장기 색상)
   - Hero 배경: 빨강(#BC002D) + 화이트 그라데이션
   - 느낌: 우아하고 세련된 느낌

4. **중국어 테마**
   - 색상: 레드 + 골드 (오성홍기 색상)
   - Hero 배경: 빨강(#DE2910) + 금색(#FFDE00) 그라데이션
   - 느낌: 럭셔리하고 화려한 느낌

#### UI/UX 개선
- **언어 선택기 재디자인**
  - 위치: 헤더 우측 상단 (네비게이션 내)
  - 디자인: 현재 언어를 국기 아이콘 + 언어명으로 표시
  - 드롭다운: 클릭 시 다른 언어 옵션 표시
  - 호버 효과: 각 언어별 색상으로 하이라이트
  - 활성 상태: 현재 선택된 언어는 골드 배경으로 강조

#### 기술 구현
- **자동 언어 감지**
  - 브라우저 언어 자동 인식 (`navigator.language`)
  - 첫 방문 시 사용자 언어로 자동 설정

- **언어 저장**
  - `localStorage`에 선택 언어 저장
  - 다음 방문 시 자동으로 저장된 언어 적용

- **실시간 번역**
  - 페이지 새로고침 없이 즉시 적용
  - `data-i18n` 속성을 활용한 동적 번역
  - 모든 텍스트 요소 자동 변환

#### 번역 적용 범위
- ✅ 네비게이션 (Nav): 모든 메뉴 항목
- ✅ 히어로 섹션 (Hero): 타이틀, 설명, CTA 버튼, 통계
- ✅ 특징 섹션 (Features): 6가지 강점 제목 및 설명
- ✅ 서비스 섹션 (Services): 4가지 서비스 제목, 설명, 기능 목록
- ✅ 포트폴리오 섹션 (Portfolio): 탭 이름
- ✅ 가격 안내 (Pricing): 패키지 정보
- ✅ 회사 소개 (About): 대표 인사말, 회사 정보
- ✅ 문의하기 (Contact): 폼 라벨, 플레이스홀더, 버튼
- ✅ 푸터 (Footer): 링크 및 저작권 정보
