## 📤 6단계: 배포하기

### Publish 탭에서 배포 (가장 쉬움)
1. 젠스파크 플랫폼의 **Publish 탭** 클릭
2. "Publish" 버튼 클릭
3. 자동으로 배포됨
4. 라이브 URL 받기

### GitHub Pages로 배포
```bash
# 1. GitHub 저장소 생성
git init
git add .
git commit -m "Initial commit"
git remote add origin YOUR_REPO_URL
git push -u origin main

# 2. GitHub Settings → Pages
# - Source: Deploy from a branch
# - Branch: main, / (root)
# - Save

# 3. 5분 후 URL 확인
https://YOUR_USERNAME.github.io/YOUR_REPO_NAME
```

### Netlify로 배포
1. [Netlify](https://www.netlify.com/) 접속 및 가입
2. "Add new site" → "Deploy manually"
3. 프로젝트 폴더 드래그 앤 드롭
4. 자동 배포 완료
5. 라이브 URL 받기

## 🚨 문제 해결

### Q: 이미지가 안 보여요
A: 이미지 URL이 유효한지 확인하세요. 브라우저 개발자 도구(F12)에서 에러 확인.

### Q: 동영상이 재생 안 돼요
A: YouTube URL이 `embed` 형식인지 확인하세요.
```
❌ https://www.youtube.com/watch?v=VIDEO_ID
✅ https://www.youtube.com/embed/VIDEO_ID
```

### Q: 모바일에서 레이아웃이 깨져요
A: 브라우저 캐시를 지우고 새로고침하세요. (`Ctrl+Shift+R`)

### Q: 폼 제출이 안 돼요
A: 현재는 프론트엔드만 구현되어 있습니다. 백엔드 API 연동이 필요합니다.
