## 🎯 프로젝트 소개

**지혜로운 손길**은 65세 시니어 대표가 운영하는 혁신적인 홈페이지 제작 전문 기업입니다. 
오랜 인생 경험과 최첨단 AI 기술(젠스파크)을 결합하여, 고객의 진짜 니즈를 이해하고 
사업의 본질을 담아내는 웹사이트를 제작합니다.

### 🌟 차별화 포인트

- **65년의 경험**: 고객의 진짜 니즈를 이해하는 깊이
- **AI 기술 활용**: 젠스파크로 3-5배 빠른 제작 속도
- **따뜻한 소통**: 눈높이에 맞춘 친절한 상담
- **합리적 가격**: 경쟁사 대비 20-30% 저렴
- **컬러풀 디자인**: 현대적이고 생동감 있는 비주얼

## ✨ 주요 기능

### 🎨 완성된 기능

#### 1. **반응형 웹 디자인**
   - PC, 태블릿, 모바일 완벽 대응
   - 1024px, 768px, 480px 브레이크포인트

#### 2. **인터랙티브 네비게이션**
   - 고정 헤더 (스크롤 시 그림자 효과)
   - 모바일 햄버거 메뉴
   - 활성 섹션 자동 표시
   - 부드러운 스크롤 애니메이션

#### 3. **히어로 섹션**
   - 임팩트 있는 메인 비주얼
   - 통계 카운터 애니메이션 (100+, 98%, 24h)
   - CTA 버튼 (무료 상담, 포트폴리오)

#### 4. **다국어 지원 (i18n)** 🌏 (NEW!)
   - 4개 언어 완전 지원: 한국어, 영어, 일본어, 중국어
   - 언어별 창의적 테마 색상 및 디자인 차별화
   - 헤더 드롭다운 언어 선택기 (국기 아이콘)
   - 브라우저 언어 자동 감지 & localStorage 저장
   - 실시간 번역 적용 (페이지 새로고침 불필요)
   - 전체 섹션 다국어 번역: Nav, Hero, Features, Services, Portfolio, Pricing, About, Contact, Footer
   - 상세 가이드: [`MULTILANG-GUIDE.md`](MULTILANG-GUIDE.md)

#### 5. **서비스 소개**
   - 6가지 차별화 강점 카드 (다국어 지원)
   - 4가지 서비스 패키지 (다국어 지원)
   - 호버 애니메이션 효과

#### 6. **포트폴리오 갤러리**
   - 이미지 포트폴리오: 6개 제작 사례 그리드
   - 동영상 포트폴리오: 6개 실제 YouTube 영상 삽입
   - 탭 전환 기능 (이미지/동영상)
   - 카테고리별 분류
   - 인터랙티브 호버 효과
   - 실제 웹 디자인 쇼케이스 영상

#### 7. **가격 안내**
   - 3가지 패키지 (기본/표준/프리미엄)
   - 투명한 가격 정책
   - 추가 서비스 안내
   - 인기 패키지 강조

#### 8. **회사 소개**
   - 대표 스토리
   - 회사 정보 (사업자번호, 주소 등)
   - 핵심 가치 소개

#### 9. **문의 폼**
   - 실시간 폼 검증
   - 전화번호 자동 포맷팅
   - 성공 메시지 표시
   - 개인정보 동의 체크

#### 10. **회원 시스템 & 데이터베이스** ✨ (NEW!)
   - **데이터베이스 테이블**: 
     - `members` (6 필드): userId, name, email, phone, password
     - `inquiries` (7 필드): name, phone, email, service, message, status
   - **RESTful Table API** 완전 통합
   - **자동 저장**: 회원가입/문의 폼 제출 시 자동으로 DB에 저장
   - **관리자 대시보드** (`admin.html`):
     - 실시간 회원 목록 조회 (이름, 이메일, 전화, 가입일)
     - 실시간 문의 목록 조회 (이름, 연락처, 서비스, 상태)
     - 탭 전환 (회원 관리 / 문의 관리)
     - 새로고침 버튼
     - 반응형 디자인

#### 11. **회원 인증 페이지**
   - 로그인 페이지 (이메일 로그인)
   - 회원가입 페이지 (실시간 유효성 검사)
   - 소셜 로그인 연동 준비 (Google, Kakao, Naver)
   - 비밀번호 표시/숨김 토글
   - 자동 로그인 & 비밀번호 찾기 기능

#### 12. **부가 기능**
   - Scroll to Top 버튼
   - 스크롤 애니메이션
   - 스크롤 이벤트 쓰로틀링 (성능 최적화)
   - 소셜 미디어 링크
   - 포트폴리오 탭 전환 (이미지/동영상)
   - YouTube 동영상 임베드

## 🚀 설치 및 실행

### 방법 1: 직접 실행 (권장)

```bash
# 1. 프로젝트 클론 또는 다운로드
git clone [repository-url]
cd wise-hands-website

# 2. index.html 파일을 브라우저에서 열기
# - 더블클릭으로 열기
# - 또는 Live Server 사용 (VS Code 확장)
```

### 방법 2: 로컬 서버 실행

```bash
# Python 3가 설치되어 있는 경우
python -m http.server 8000

# 브라우저에서 접속
# http://localhost:8000
```

### 방법 3: VS Code Live Server

1. VS Code에서 프로젝트 열기
2. Live Server 확장 설치
3. `index.html` 우클릭 → "Open with Live Server"
