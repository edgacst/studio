## 📱 언어 선택기 사용법

### 1️⃣ 위치
- **헤더 오른쪽 끝**에 위치
- 로그인 버튼 옆
- 모든 페이지에서 동일한 위치

### 2️⃣ 디자인
```
┌─────────────────────────────────────┐
│ 🇰🇷 KO | 🇺🇸 EN | 🇯🇵 JA | 🇨🇳 ZH │
└─────────────────────────────────────┘
```
- **반투명 배경**: 고급스러운 느낌
- **국기 아이콘**: 직관적인 선택
- **활성 언어**: 금색으로 강조
- **호버 효과**: 부드러운 애니메이션

### 3️⃣ 사용 방법
1. 원하는 언어 버튼 클릭
2. 페이지 전체가 즉시 번역
3. 선택한 언어가 로컬스토리지에 자동 저장
4. 다음 방문 시 저장된 언어로 자동 표시

## 🔧 기술 구현

### JavaScript 파일
- **js/i18n.js** (29KB)
  - 4개국 완전 번역 데이터
  - 자동 언어 감지
  - 로컬스토리지 저장

### HTML 속성
```html
<!-- 기본 사용 -->
<h1 data-i18n="hero.title">지혜로운 손길</h1>

<!-- 플레이스홀더 번역 -->
<input data-i18n-placeholder="contact.name_placeholder">

<!-- 버튼 값 번역 -->
<input type="submit" data-i18n="contact.submit_button">
```

### CSS 스타일
```css
/* 언어 선택기 컨테이너 */
.language-selector {
    background: rgba(255, 255, 255, 0.05);
    padding: 4px;
    border-radius: 12px;
}

/* 언어 버튼 */
.lang-btn {
    padding: 8px 12px;
    border-radius: 8px;
    transition: all 0.3s ease;
}

/* 활성 언어 */
.lang-btn.active {
    background: var(--secondary-color);
    color: var(--dark-color);
}
```

## 🌐 브라우저 자동 감지

### 지원 언어 감지
```javascript
const browserLang = navigator.language.split('-')[0];
// 'ko-KR' → 'ko'
// 'en-US' → 'en'
// 'ja-JP' → 'ja'
// 'zh-CN' → 'zh'
```

### 우선순위
1. **로컬스토리지 저장값** (사용자가 선택한 언어)
2. **브라우저 언어** (자동 감지)
3. **기본 언어** (한국어 - ko)


```
5. 다음 방문 시 자동으로 일본어 표시
```
