## ⚙️ 기술적 세부사항

### 번역 시스템 구조

```javascript
// 번역 데이터 구조
translations = {
    ko: { nav: {...}, hero: {...}, features: {...}, ... },
    en: { nav: {...}, hero: {...}, features: {...}, ... },
    ja: { nav: {...}, hero: {...}, features: {...}, ... },
    zh: { nav: {...}, hero: {...}, features: {...}, ... }
}
```

### 번역 적용 방식

1. **HTML 마크업**
   ```html
   <h1 data-i18n="hero.title">당신의 성공을 위한</h1>
   ```

2. **JavaScript 자동 번역**
   ```javascript
   function translatePage() {
       document.querySelectorAll('[data-i18n]').forEach(element => {
           const key = element.getAttribute('data-i18n');
           const translation = getTranslation(key);
           element.textContent = translation;
       });
   }
   ```

3. **언어 자동 감지**
   - 브라우저 언어 자동 감지
   - localStorage에서 이전 선택 언어 복원
   - 기본값: 한국어(ko)
